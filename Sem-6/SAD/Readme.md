**CANVAS** is a premium, web-based booking platform designed for Ahmedabad's creative community. It provides an intuitive interface for creators, performers, and podcasters to book themed, high-fidelity studio spaces by the hour.

## 🚀 Project Introduction
The platform addresses the lack of accessible, themed creative venues by offering five specialized rooms: Traditional (Heritage), Disco/Neon, Podcast Studio, Workshop/Rehearsal, and a fully customizable Convertible Room. Built with a focus on seamless UX, it handles everything from real-time slot discovery to secure payment processing.

## 🛠️ System Design & Architecture
The system is designed as a lightweight, high-performance Single Page Application (SPA) architecture:

* **Frontend**: Built using semantic HTML5, CSS3 (utilizing CSS variables and Flexbox/Grid for responsiveness), and Vanilla JavaScript.
* **State Management**: Implements a centralized `Canvas` utility object in `app.js` to manage authentication, session persistence via `localStorage`, and booking logic.
* **Workflow**: Features a 3-step secure booking funnel (Details → Review → Payment) with integrated real-time validation and simulated payment gateways.
* **Design System**: Follows a "Dark Luxury" aesthetic using a palette of Gold (#c9a84c) and Charcoal (#0d0d0d), emphasizing high-fidelity visual feedback (shimmer effects and page transitions).

## 📁 Key Features
* **Themed Discovery**: Filterable room gallery based on use-cases (Shoots, Events, Recording).
* **Dynamic Booking**: Real-time date and slot selection with automatic GST and fee calculations.
* **User Dashboard**: Personal "My Bookings" area to track upcoming sessions and download receipts.
* **Secure Checkout**: Multi-method payment interface (UPI, Card, Net Banking).
