1. AIM

To perform basic file and folder operations like creating, deleting, renaming, moving, adding content, and deleting content using Command Prompt.

2. THEORY

The Command Prompt (CMD) is a command-line interpreter in the Windows operating system. It allows users to execute commands to perform various tasks, including managing files and directories. Common commands used for file and folder operations include:

  • mkdir: To create folders
  • del: To delete files
  • rmdir: To delete folders
  • rename: To rename files or folders
  • move: To move files or folders
  • echo: To add content to files

These commands are integral for automating tasks, managing file systems, and enhancing productivity.

3. PROCEDURE

  1. Create a Folder: 
    mkdir leo
  2. Create a File: 
    echo. > leo.txt
  3. Add Content to the File: 
    echo Kavya Trivedi is legend > leo.txt
  4. Delete Content from the File: 
    echo. > leo.txt
  5. Rename the File: 
    rename leo.txt kavyaleo.txt
  6. Move the File: 
    kavyaleo.txt move c::\user\kavya\kavyatrivedi
  7. Delete the File: 
    del kavyaleo.txt
